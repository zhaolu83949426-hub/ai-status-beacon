# Product

## Register

product

## Users
使用 Codex、Claude Code、Gemini CLI 等 AI Agent 的开发者。用户在桌面环境中持续工作，需要快速知道 Agent 当前是空闲、执行中、等待审批还是异常。

## Product Purpose
AI Status Beacon 是一个桌面状态灯工具，把多个 AI Agent 的执行状态、审批请求和额度信息集中显示在状态栏与管理窗口中。成功标准是状态一眼可辨、审批入口可靠、设置过程清晰。

## Brand Personality
安静、精确、可信。界面应像桌面系统工具一样克制，同时让状态灯成为明确的视觉焦点。

## Anti-references
避免营销页式大标题、装饰性卡片堆叠、低对比灰字、过度玻璃拟态，以及会干扰工作流的夸张动效。

## Design Principles
状态优先：颜色和发光强度只服务于 Agent 状态判断。
桌面原生：控件、间距和反馈遵循熟悉的工具型产品习惯。
信息安静：设置与列表页面保持低噪声，让用户快速扫读。
失败可见：审批、Hook、额度和错误状态必须清楚呈现。

## Accessibility & Inclusion
默认遵循 WCAG AA 对比度要求。状态颜色应配合文字标签或位置关系表达，不能只依赖颜色；动效需尊重减少动态偏好。
